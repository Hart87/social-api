Social API
Users Posts Auth

Base Route: /users


Users Schema

Long : id - Primary Key
String : createdAt
String : updatedAt
String : email - Not Null
String : username - Not Null
String : password - Not Null
String : role - Not Null

ID is generated automatically by Postgres. The times for createdAt & updatedAt are generated and set in the Users controller under the POST & PUT routes. Times are generated in current milliseconds

Presently there is no auth on POST of /users. Soon there will be an I.P. Address based rate limiter on this route to prevent malicious actions against this route.

Users Routes

All routes accept and return JSON
GET - "/users/{id} Get a User by ID number
GET - "/users/all Get all of the users
POST - "/users Create a user. Takes in a request body of User
PUT - "/users/{id} Update a user. Takes in a request body of User and a Path Variable of a users ID
DELETE - "/users/{id} Delete a user by ID number
