About


The Social API is a mock social network API written in Java with Spring Boot. Persistence is achieved by a Postgres Database ran in a docker container alongside the API. It can easily be run with Docker Compose. Authentication is provided by the used of JWT(okens) which are good for one hour. Routes are available through the entity names in the navbar.

To run the project : You will need Docker. CD into the src/main/docker folder. In there is a Dockerfile for the server and a docker-compose.yml to run the server and the database together. While in the folder run the command : docker compose-up and it should be available at http://localhost:8081

This is an ongoing project and these docs will be updated accordingly. Rate Limiting and Caching with Redis will be added in the future. Any pull request please submit on a separate branch.
