Social API
Users Posts

Base Route: /posts


Posts Schema

Long : id - Primary Key
String : createdAt
String : updatedAt
Long : userId - Not Null
String : text - Not Null
Integer : likes

ID is generated automatically by Postgres. The times for createdAt & updatedAt are generated and set in the Posts controller under the POST & PUT routes. Times are generated in current milliseconds

Posts Routes

All routes accept and return JSON - no auth yet.
GET - "/posts/{id} Get a post by ID number
GET - "/posts/{user_id}/all Get all of the posts by a specific user
GET - "/posts/all Get all of the posts by a specific user
POST - "/posts Create a post. Takes in a request body of Post
PUT - "/posts/{id} Update a post. Takes in a request body of Post and a Path Variable of a post ID
PUT - "/posts/{id}/like Update a post by increasing the likes count.
PUT - "/posts/{id}/unlike Update a post by decreasing the likes count.
DELETE - "/posts/{id} Delete a post by ID number
